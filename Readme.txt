Database Configuration
1. create database as testdb in MySQL workbench
	create database testdb;
	use testdb;

For Backend
1. Open the backend project in Spring Boot Tool Suite
2. Export the project as Existing Maven Project
3. Then run the project as Java Server Application

for front End

Please use following command in command prompt in vs code
1. To install the project
	npm install
2. To run the project
	npm start
